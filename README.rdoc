= cpspec
Cpspec is a framework which allows behaviour driven development for chef cookbooks and puppet manifests. Its dependency is on vagrant and hence Oracle virtual box.

= Installation
* Install vagrant. Installation instructions can be found here.(http://vagrantup.com/docs/getting-started/index.html)

 gem install cpsec
